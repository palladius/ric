* [ric](http://rubygems.org/gems/ric)
* [Riclib](http://github.com/palladius/riclib)

## Installation

To use Ric in a Rails 3 application:

1. Require it in the Gemfile: `gem 'ric'`

2. Install it by running `bundle install`.
